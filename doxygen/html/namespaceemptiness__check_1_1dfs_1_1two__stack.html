Look for the accepting lasso in NBA/NGA automaton.

While the state is grey, the algorithm tries to find a cycle containing it. If it succeeds, then the state is removed from C. If not, then the state is removed from C when it is blackened. At any time t, the candidates are the currently grey states that do not belong to any cycle of At.
